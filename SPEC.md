# ReceiptLift — MVP Landing Page

Receipt capture → auto-categorize → one-click accountant export.
$9/month solo, $19/month business. Built for owner-operators who don't want full bookkeeping software.

## Files
- `/app/` — Next.js app router structure
- `/app/page.tsx` — Main landing page
- `/app/layout.tsx` — Root layout with providers
- `/components/` — shadcn/ui components
- `/app/globals.css` — Tailwind + custom CSS variables

## Stack
- Next.js 14+ (App Router)
- shadcn/ui components
- Tailwind CSS
- Lucide React icons
- next-themes for dark mode

## Design
- Primary color: #059669 (emerald green — trust, money, growth)
- Accent: #f59e0b (amber — receipt warmth, attention)
- Clean, professional, mobile-first
- No animations needed — just clean layout and clear copy

## Features on Page
- Hero with product screenshot/demo
- Waitlist CTA (localStorage for now, backend later)
- Pricing section ($9/$19/month)
- Feature highlights
- Social proof placeholder
